💡 HOW TO USE THE POWERFUL FILE MANAGEMENT:

STEP 1: SCAN YOUR FOLDER
  → Click "🔍 SCAN FOLDER" button in sidebar
  → Select a folder (e.g., Downloads, Documents, Desktop, etc.)
  → App shows: Total files, storage used, file types, recent files

STEP 2: VIEW FILES IN TABLE
  → Go to "📁 Files" tab
  → See all files in professional table format
  → Columns: Name, Size, Type, Modified, Path

STEP 3: FILTER FILES
  You have multiple ways to filter:
  
  a) SEARCH BY NAME:
     → Type in search box (real-time filtering)
     → Shows only files matching your search
  
  b) FILTER BY CATEGORY:
     → Select from dropdown:
        • All Files - No filter
        • Documents - .pdf, .docx, .xlsx, etc.
        • Images - .jpg, .png, .gif, etc.
        • Videos - .mp4, .avi, .mkv, etc.
        • Audio - .mp3, .wav, .flac, etc.
        • Archives - .zip, .rar, .7z, etc.
        • Code - .py, .js, .java, etc.
        • Large Files - Files above size limit
        • Duplicates - Identical files
  
  c) FILTER BY SIZE:
     → Set minimum size in MB
     → Shows only files above that size
  
  d) COMBINE ALL FILTERS:
     → Search + Category + Size all work together!
     → Perfect for finding specific files

STEP 4: MANAGE FILES
  → Click on a file in the table
  → Choose action:
     • 📂 Open File - Open directly
     • 🗑️ Delete - Remove file
     • 📋 Copy Path - Copy to clipboard

STEP 5: ORGANIZE YOUR FOLDER
  → Go to "📋 Organization" tab
  → Choose organization method:
  
  a) AUTO-ORGANIZE BY TYPE:
     → Creates folders: Documents, Images, Videos, Audio, Archives, Other
     → Automatically moves files
     → Confirmation dialog before proceeding
  
  b) ORGANIZE BY DATE:
     → Creates structure: 2024/10/16/
     → Organizes by file modification date
     → Great for archival
  
  c) FIND DUPLICATES:
     → Scans folder for identical files
     → Uses size + MD5 hash
     → Shows duplicate groups
  
  d) CLEANUP EMPTY FOLDERS:
     → Removes all empty directories
     → Recursive cleanup
     → Shows count removed

STEP 6: VIEW ANALYTICS
  → Go to "📈 Analytics" tab
  → See detailed statistics:
     • Total Files
     • Total Storage
     • File Types
     • Largest File
     • Average File Size

STEP 7: TOGGLE DARK MODE
  → Press Ctrl+D anytime
  → OR go to View → Toggle Theme
  → Theme changes everywhere instantly
