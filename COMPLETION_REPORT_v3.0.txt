📋 USAGE INSTRUCTIONS:

Quick Start:
  1. python main.py (or double-click RUN_APP.bat)
  2. Dark mode ON by default 🌙
  3. Click "🔍 SCAN FOLDER"
  4. Select any folder
  5. Explore tabs!

Toggle Theme:
  • Press Ctrl+D anytime
  • Or go to View → Toggle Theme

Organize Files:
  1. Scan folder
  2. Go to Organization tab
  3. Choose method (type, date, etc.)
  4. Confirm action
  5. Done! ✅

Find Files:
  1. Go to Files tab
  2. Search by name
  3. Filter by category
  4. Filter by size
  5. Review results
