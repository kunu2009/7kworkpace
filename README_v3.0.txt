🚀 GETTING STARTED:

STEP 1: Launch the App
  • Double-click RUN_APP.bat, OR
  • Type: python main.py, OR
  • Use Command Prompt/Terminal

STEP 2: You're in Dark Mode by Default
  • Professional, easy on the eyes 🌙
  • Press Ctrl+D to toggle to light mode

STEP 3: Scan Your Folder
  • Click "🔍 SCAN FOLDER" button
  • Select any folder (Downloads, Documents, Desktop, etc.)
  • App analyzes all files instantly

STEP 4: Explore Tabs
  • 📊 Dashboard: See overview and statistics
  • 📁 Files: Advanced file browser with filters
  • 📋 Organization: Tools to organize your folder
  • 📈 Analytics: Detailed statistics
  • 📝 Notes: Create and save notes
  • 📅 Calendar: Plan your activities

STEP 5: Try Filtering
  • Search by name
  • Filter by category (Documents, Images, etc.)
  • Filter by size
  • Combine all three!

STEP 6: Organize
  • Go to Organization tab
  • Choose organization method
  • Confirm action
  • Done! ✅

⚡ PERFORMANCE TIPS:

✓ Best Performance:
  • Folders with < 100,000 files
  • Local drives (not network drives)
  • Use filters to reduce items shown
  • Search + category filter together

✓ For Large Folders:
  • Organize by Date first
  • Then organize by type within each date folder
  • This splits large folder into smaller ones

✓ Display Performance:
  • Table shows first 500 files
  • Filtering happens instantly
  • Scrolling is smooth
  • No lag in dark mode
