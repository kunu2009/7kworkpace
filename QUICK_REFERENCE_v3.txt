🎯 COMMON WORKFLOWS

Organize Downloads Folder:
  1. Scan Folder → Select Downloads
  2. Organization → Auto-Organize
  3. See results in Analytics

Find Large Files:
  1. Scan Folder
  2. Files Tab → Set Min Size to 500 MB
  3. See files larger than 500 MB

Cleanup Duplicates:
  1. Scan Folder
  2. Organization → Find Duplicates
  3. Review and manually delete
  4. Organization → Cleanup Empty

✅ QUICK START

1. Launch: python main.py or double-click RUN_APP.bat
2. Dark mode is ON by default 🌙
3. Click "🔍 SCAN FOLDER"
4. Select your folder
5. Browse files in "📁 Files" tab
6. Use "📋 Organization" tab to organize
7. Check "📈 Analytics" for stats
8. Press Ctrl+D to toggle light/dark mode

❓ FREQUENTLY ASKED

Q: How do I turn off dark mode?
A: Press Ctrl+D or go to View → Toggle Theme

Q: Can I delete files?
A: Yes! Select file in Files tab, click Delete (confirmation required)

Q: How do I organize my folder?
A: Use Organization tab - Auto-Organize or Organize by Date

Q: Where are my notes saved?
A: C:\Users\[YourUsername]\.workspace_organizer\notes\

Q: How do I find duplicate files?
A: Organization tab → Find Duplicates

Q: Can I undo organizing?
A: No - be careful! Use backup of your folder first

Q: Does it work with network drives?
A: Yes, but slower - use local drives when possible
