Best Performance:

🆘 TROUBLESHOOTING QUICK FIXES:

Problem: "Python not found"
Solution: Install Python from https://www.python.org/
          Add to PATH, restart, try again

Problem: "Module not found"
Solution: Run: python -m pip install -r requirements.txt
          Wait for completion, try again

Problem: App won't start
Solution: Double-click RUN_APP.bat instead
          Check Command Prompt for errors
          Make sure all files in correct folders

Problem: Build fails
Solution: python -m pip install pyinstaller
          Try building again

Problem: "Permission denied"
Solution: Right-click Command Prompt
          Run as Administrator
          Try again

For more help: Read SETUP_GUIDE.txt
