❓ NEED HELP?

First:
  → Check SETUP_GUIDE.txt for troubleshooting
  → Read error messages carefully
  → Search error online

Common Issues:

"Python not found"
  → Install Python from: https://www.python.org/
  → Add Python to PATH
  → Restart computer

"Module not found"
  → Run: python -m pip install -r requirements.txt
  → Wait for completion
  → Try again

"App won't run"
  → Double-click RUN_APP.bat (easier!)
  → Check Command Prompt for error messages
  → Make sure all files are in correct folders
