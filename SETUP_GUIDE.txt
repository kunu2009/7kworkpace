✅ STEP 1: INSTALLATION

A. Prerequisites:
   ✓ Windows 10 or Windows 11
   ✓ Python 3.8 or higher (Download from: https://www.python.org/downloads/)
   
   To check if Python is installed:
   - Open Command Prompt (Win + R, type: cmd)
   - Type: python --version
   - You should see: Python 3.x.x

B. Install Dependencies:
   Method 1 (Easiest - Just double-click):
   ✓ Double-click: RUN_APP.bat
   ✓ It will automatically install everything needed
   
   Method 2 (Manual - via Command Prompt):
   ✓ Open Command Prompt in this folder
   ✓ Type: python -m pip install -r requirements.txt
   ✓ Wait for installation to complete

📦 STEP 3: BUILDING THE EXECUTABLE (.exe)

What is an executable?
   - A standalone .exe file that works without needing Python installed
   - You can share it with friends or run it on any Windows computer
   - No dependencies needed on the target computer

How to build:

Method 1 (Easiest - Just double-click):
   ✓ Double-click: BUILD_EXE.bat
   ✓ Wait for the build to complete (3-5 minutes)
   ✓ Your .exe will be in the "dist" folder

Method 2 (Using Command Prompt):
   ✓ Open Command Prompt in this folder
   ✓ Type: python build_exe.py
   ✓ Wait for build to complete

After building:
   ✓ Find the executable: dist/WorkspaceOrganizer.exe
   ✓ Copy it to your Desktop
   ✓ Create a shortcut (right-click > Send to > Desktop)
   ✓ You can now run it anytime!

🆘 TROUBLESHOOTING

Problem: "Python is not installed"
Solution:
   • Download Python from: https://www.python.org/downloads/
   • During installation, CHECK: "Add Python to PATH"
   • Restart Command Prompt
   • Try again

Problem: "Command not found" or "python not found"
Solution:
   • Make sure Python is installed and added to PATH
   • Try: py main.py (instead of python main.py)
   • Or restart your computer

Problem: "ModuleNotFoundError: No module named 'PyQt6'"
Solution:
   • Dependencies not installed
   • Run: python -m pip install -r requirements.txt
   • Wait for completion
   • Try running the app again

Problem: App crashes or won't start
Solution:
   • Check error message in Command Prompt
   • Make sure all dependencies are installed
   • Try: python -m pip install --upgrade PyQt6
   • Restart the app

Problem: Executable won't build
Solution:
   • Make sure PyInstaller is installed:
     python -m pip install pyinstaller
   • Try building again
   • If still fails, check console for detailed error

Problem: "Permission denied" when scanning folder
Solution:
   • Right-click Command Prompt
   • Select "Run as administrator"
   • Try scanning again

💡 TIPS & TRICKS

• The app works best with folders containing 10,000 files or less
• Notes are saved automatically when you create them
• Avoid scanning system folders (C:\Windows) as they're huge
• For best performance, use with standard user folders:
  - C:\Users\YourUsername\Documents
  - C:\Users\YourUsername\Pictures
  - C:\Users\YourUsername\Desktop
