💡 TIPS FOR BEST PERFORMANCE:

✓ Scan folders with <10,000 files for best performance
✓ Use common user folders:
   • Documents
   • Pictures
   • Desktop
   • Downloads
✓ Avoid system folders (C:\Windows)
✓ Close the app before major file changes
✓ Periodically clean up old notes
✓ For very large folders, consider splitting into subfolders

🆘 NEED HELP?

Issue: "Python not found"
→ Install Python from: https://www.python.org/
→ Make sure to check "Add Python to PATH"

Issue: Dependencies not installing
→ Open Command Prompt
→ Type: python -m pip install --upgrade pip
→ Then: python -m pip install -r requirements.txt

Issue: App won't start
→ Double-click RUN_APP.bat (easier!)
→ Or check error in Command Prompt
→ Make sure all files are in the right folders

Issue: Build fails
→ Run: python -m pip install pyinstaller
→ Try building again

See SETUP_GUIDE.txt for more troubleshooting!
