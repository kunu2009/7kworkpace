🔍 FILTER SYSTEM (Files Tab):

Search Box:
  ┌─ Real-time filtering
  ├─ As you type, results update
  ├─ Case-insensitive
  └─ Works with other filters

Category Filter (Dropdown):
  ┌─ All Files (no filter)
  ├─ Documents (.pdf, .doc, .docx, etc.)
  ├─ Images (.jpg, .png, .gif, etc.)
  ├─ Videos (.mp4, .avi, .mkv, etc.)
  ├─ Audio (.mp3, .wav, .flac, etc.)
  ├─ Archives (.zip, .rar, .7z, etc.)
  ├─ Code (.py, .js, .java, etc.)
  └─ (All filters work together!)

Size Filter (MB):
  ┌─ Minimum file size in MB
  ├─ 0 = no filter
  ├─ 100 = 100+ MB files
  ├─ 500 = 500+ MB files
  └─ (Works with other filters!)

💾 WHAT HAPPENS WHERE:

Scanned Files:     Displayed in Files tab, Dashboard recent files
Organized Files:   New folders created in scanned folder
Duplicates:        Listed in results (you decide what to do)
Empty Folders:     Removed by cleanup tool
Notes:             Saved to ~/.workspace_organizer/notes/
Settings:          Saved to ~/.workspace_organizer/config.json
