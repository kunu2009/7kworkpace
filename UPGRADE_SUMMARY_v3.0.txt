🎯 HOW TO USE:

QUICK START (30 seconds):
  1. Launch: python main.py (or double-click RUN_APP.bat)
  2. Click "🔍 SCAN FOLDER"
  3. Select a folder (e.g., Downloads)
  4. See results in dashboard
  5. Explore Files, Organization, Analytics tabs
  6. Press Ctrl+D to toggle theme

ORGANIZE YOUR FOLDER (2 minutes):
  1. Scan your folder
  2. Go to "📋 Organization" tab
  3. Choose organization method:
     - Auto-Organize → Creates type folders
     - Organize by Date → Creates date structure
     - Find Duplicates → Detects copies
     - Cleanup → Removes empty folders
  4. Click button and confirm
  5. Done! ✅

FIND SPECIFIC FILES (1 minute):
  1. Scan your folder
  2. Go to "📁 Files" tab
  3. Use search + filters:
     - Search by name
     - Filter by category
     - Filter by size
  4. Review results in table
  5. Open/Delete/Copy files as needed
